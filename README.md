# Log Detective

AI-powered real-time error log monitoring and analysis agent.

## Overview

Log Detective is a CLI tool that monitors log files in real-time, detects errors, and provides AI-powered root cause analysis with solution suggestions.

## Features (Planned)

- **Log Parsing**: Support for Java, Python (and more) log formats
- **Real-time Monitoring**: Watch log files and detect errors automatically
- **RAG-based Analysis**: Search related source code and similar past errors
- **AI Agent**: Intelligent analysis workflow using LangGraph
- **Slack Alerts**: Get notified when errors occur

## Tech Stack

- Python 3.11+
- LangGraph / LangChain
- Chroma (Vector DB)
- FastAPI (optional API server)
- OpenAI / Claude API
